# 🛒 Modern Webshop API Template

This is a high-performance, scalable webshop backend template built with **.NET 10 Minimal API**. It provides a production-ready foundation for e-commerce applications, focusing on clean architecture, security, and ease of customization.

## 🛠️ Tech Stack

| Layer | Technology | Key Usage |
| :--- | :--- | :--- |
| **Framework** | **.NET 10 (C#)** | Minimal API for high performance and low overhead. |
| **Database** | **PostgreSQL** | Reliable relational data storage. |
| **ORM** | **EF Core** | Database management and complex queries with LINQ. |
| **Security** | **JWT & CORS** | Secure, role-based access control and frontend integration. |
| **Mailing** | **MailKit** | SMTP integration for security codes and notifications. |
| **Configuration** | **dotenv.net** | Secure management of environment variables. |
